## Running the project

Run the Python script:
```
python hello_world.py
```

Open the HTML page by opening `index.html` directly in a browser — no build step or server required.

## Project structure

This is a minimal two-file Hello World project for Dezernat Zukunft (Institut für Makrofinanzen):

- [hello_world.py](hello_world.py) — prints "Hello World" to stdout
- [index.html](index.html) — a styled static HTML page with Dezernat Zukunft branding (dark green accent `#1e4d35`, Inter font, German-language copy)
